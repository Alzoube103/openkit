## üé® Using openkit
Once installed, you can start creating applications with openkit. Here are some steps to help you begin:

1. **Open the Application**: Launch openkit from your applications folder.
2. **Create a New Project**: Start a new project by selecting "New Project" from the main menu.
3. **Choose a Template**: Select a template that fits your needs. You can start with a blank project or use a predefined style.
4. **Edit the Style**: Use CSS to customize the look of your application. Modify styles in the provided CSS editor.
5. **Add Widgets**: Drag and drop from over 30 available widgets to create your interface. You can add buttons, text boxes, and more.
6. **Preview Your App**: Use the preview feature to see how your application looks in real-time.
7. **Build and Run**: When you are satisfied, click the "Run" button to test your application.
